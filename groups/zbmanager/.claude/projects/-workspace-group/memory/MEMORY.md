## Структура zbpages

### Финансы
Путь: `/workspace/extra/zbpages/Управление/Финансы/БДР/`

Основные файлы:
- `бдр янв_фев 2026.xls` - бюджет доходов и расходов за текущий год
- `БДР янв_дек 2024.xls`, `БДР янв_дек 2025_.xls` - архив за прошлые годы
- `Анализ движений денежных средств за Февраль 2026 г.xlsx` - детализация по расчётным счетам
- `Зарплата_февр_26.pdf` - расчётная ведомость по зарплате

### Как читать БДР
- Используй pandas: `pd.read_excel()`
- В БДР строка 4 (индекс 4) - заголовки: "Показатель", "Всего", "Январь 2026", "Февраль 2026" и т.д.
- Данные начинаются со строки 5 (индекс 5)
- Основные показатели:
  - Строка 5: Выручка итого
  - Строка 7: Финансовый результат
  - Строка 9: Итого расходы
  - Строка 10: Себестоимость (продукты)
  - Строка 22: Оплата труда (ФОТ)
  - Строка 27: Коммерческие расходы
